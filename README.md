# BUPT南太孔院宣传门户开发

> BUPT_sp advertisement



## IDEA Clone事项

1. maven配置

   克隆后请检查pom.xml,如果为此图标

![image-20240313232451477](README.assets/image-20240313232451477.png)

则表明maven配置已完成，进行vue配置



2. vue配置

   检查是否有vue/node_modules

   - 如果没有，在终端执行

     ```
     cd vue
     npm i
     ```

     安装Vue

   - 如果有，进行启动配置

     > 编辑配置->  +  ->  npm  ->  package.jason(P)：选择jason  ->  脚本(T)选择：serve

   - 启动
